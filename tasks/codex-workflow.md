# Codex Workflow

This repository uses a lightweight six-step workflow.

1. Planning
   Define scope, assumptions, and the smallest verifiable change.
2. Implementation
   Make focused, reversible edits that fit existing conventions.
3. Debugging
   Check for obvious issues locally before broader validation.
4. Testing
   Run the smallest reliable verification for the current change.
5. Verification
   Record reproducible checks before claiming completion.
6. Review
   Summarize outcomes, risks, and the next follow-up work.
